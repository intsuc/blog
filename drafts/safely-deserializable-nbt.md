# Safely deserializable NBT

Not all NBTs are deserializable, i.e. can be loaded from a file.
If we try to load a deeply nested tag, we will get the following error message and the tag will not be loaded.

```
java.lang.RuntimeException: Tried to read NBT tag with too high complexity, depth > 512
```

As the error message suggests, Minecraft does not load an NBT with depth greater than 512.
Concretely, the following 513-fold list tag is deserialzable,
